Lessons

Our lessons are pretty simple, learn 30+ languages from our classrooms! All classrooms you can take they are online or in the real world. Here is some of the languages that you can learn and the difficult of the learning!

HTML and CSS - Beginners

This is one of our easiest classes you can start learning even if you never ever touched code in your whole life.

HTML and CSS - Medium

This is the second course of HTML and CSS, you will learn how to design sites that are mobile friendly and how to make some good web design.

HTML and CSS - Hard

This is the last course of HTML and CSS, you can learn how to make modern web design and how to make a decent site for the modern era of website design of 2020. Not Recommended for Beginners - Good for people that Finished JS Medium Class!

JavaScript - Beginners

This is the JavaScript class, learn JS and use your HTML and CSS skills to help yourself to make a website. Recommended for HTML and CSS - Medium Users

JavaScript - Medium

JavaScript - Hard

This is JavaScript hard, it is very hard. One of the hardest classes in our classroom. This is very not recommended for beginners or mediumists. Not Recommended for Beginners and Mediumists, Recommended for HTML and CSS - Hard users.

Lua for Beginners

Lua - Medium
